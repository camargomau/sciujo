_**(definition)**_

A [[Function|function]] $f : \mathbb{R}^n \to \mathbb{R}$ is _linear_ if and only if there are constants $c_{1}, c_{2}, \dots, c_{n}$ such that:

$$
f(x_{1}, x_{2}, \dots, x_{n}) = c_{1} x_{1} + c_{2} x_{2} + \dots + c_{n} x_{n}
$$
