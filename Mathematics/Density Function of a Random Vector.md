_**(definition)**_

Let $X, Y$ be [[Random Variable|random variables]].

We define the (joint) density function of the [[Random Vector|random vector]] $(X, Y)$ as:

$$
f_{(X, Y)} (x, y) =
\begin{cases}
\mathbb{P}(X=x, Y=y) &\text{if } (x,y) \in R_{(X,Y)} \\
0 &\text{otherwise}
\end{cases}
$$
