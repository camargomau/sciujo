A class B [[IP Address|IP addres]] is one whose range of the first octet goes from $128$ to $191$.

All class B IPs are prefixed by `10` (_==10==000000 = 128 - ==10==111111 = 191_).

Exmples of class B addresses:

- 128.0.18.0
- 172.28.47.92
- 191.255.255.255
- 163.45.99.16
